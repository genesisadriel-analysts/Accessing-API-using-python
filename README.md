# Accessing API using Python

This repository contains a Jupyter Notebook demonstrating the fundamentals of HTTP requests using the Python `requests` library.

## Key Features
* **GET Requests:** Fetching data with URL parameters.
* **POST Requests:** Sending form data to a server.
* **Response Handling:** Analyzing status codes, headers, and JSON payloads.
* **Binary Data:** Downloading and saving images locally using the `Pillow` library.

## Requirements
To run this notebook, you will need:
* Python 3.x
* `requests`
* `Pillow` (PIL)

pip install requests pillow
