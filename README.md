🌐 Accessing API using Python
📌 Project Overview

This project demonstrates how to access and interact with web APIs using Python. It focuses on retrieving data from external sources, understanding HTTP responses, and preparing data for further analysis or processing.

The notebook provides a practical introduction to working with APIs—an essential skill for modern data professionals.

🎯 Objectives
Access data from APIs using Python
Understand HTTP request and response structure
Extract useful information from API responses
Work with different data formats (HTML, JSON, images)
Build a foundation for data ingestion workflows
🧠 Skills Demonstrated
API Integration
Data Extraction from Web Sources
Response Handling & Debugging
JSON & Web Data Processing
Basic Data Transformation
🛠️ Technologies Used
Python
requests – handling HTTP requests
Pillow (PIL) – image processing
Jupyter Notebook – interactive environment
